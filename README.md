## General-info
Realisation of a free-publishing ads website with Laravel. 


## Set Up 
**Before Everything**
Move to the freeads folder.

Run the (composer install) command.

Create a new freeads database in phpmyadmin.

Create an .env file at the racine of the project and copy the contents of the .env.example file.

Modify the .env file with your database connection information.

Run the command to migrate the tables in our database freeads with this command (php artisan migrate).

Run the (npm run dev) command in a terminal.

In another terminal, run the service artisanal php command (php artisan serve).

## technologies
